Цей репозиторій містить клієнтську частину (Frontend) веб-додатка для управління розкладом занять, викладачами та студентами. Проект реалізує рольову модель доступу та базові CRUD-операції.

---

## ✨ Основні можливості

* **Система авторизації:** Вхід та реєстрація користувачів із поділом на ролі (Студент, Викладач, Адміністратор).
* **Кабінет Адміністратора:** * Керування записами викладачів та студентів (додавання, видалення) з БД.
    * Створення та редагування розкладу занять (прив'язка групи, дня тижня, пари, предмету та викладача).
* **Кабінет Студента:** Перегляд персонального розкладу занять виключно для своєї групи.
* **Кабінет Викладача:** Перегляд власного графіка викладання із зазначенням предметів та цільових груп.
* **Маршрутизація (Vue Router):** Можливість навігації між сторінками авторизації та окремими CRUD-компонентами.

## 🛠 Технологічний стек

* **Фреймворк:** Vue 3
* **Маршрутизація:** Vue Router
* **HTTP-клієнт:** Axios для взаємодії з REST API бекенду
* **Конфігурація:** Babel, налаштування Vue CLI (з відключеним лінтером для спрощення тестування)

📂 Структура Frontend-частини
src/App.vue — Головний компонент, що містить основну бізнес-логіку авторизації, взаємодії з бекендом (Axios) та відображення різних кабінетів залежно від ролі.
